# Proyecto: Sistema de Alojamiento

## Descripción

Este es un sistema sencillo de gestion de habitaciones, usuarios y registrode reservación de habitaciones hecho en Python

---

## ¿Por qué este proyecto?

Este proyecto fue creado como una práctica inicial para aprender los fundamentos de Python y su franework de desarrollo para API Rest Framework.
